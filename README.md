# express-todo
#### Author: Reymund Delos Santos III

##### Techs
* Node JS
* Express.js
* Mongo DB
* Mongoose
* Jest


##### Install Dependencies,
Simply run the command below
```terminal
npm install
```

###### Configurations: .env file
```ini
PORT=3000
MONGODB_URI=mongodb://localhost:27017/express-todo
``````

##### To start the server
Simply run the command below
```terminal
npm start
```
